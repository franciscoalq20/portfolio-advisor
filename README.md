# Portfolio Advisor 📊

Herramienta profesional de gestión multi-cartera para asesores financieros.

## ¿Qué hace?
- Seguimiento de múltiples carteras de clientes
- Semáforo visual: en objetivo / atención / desvío crítico
- Gráfico de trayectoria real vs objetivo (TEA)
- Precios en tiempo real: CoinGecko, Yahoo Finance, Bluelytics/BCRA
- Datos guardados en el navegador (sin servidor)

## Cómo usarlo

1. Abrí la URL de tu GitHub Pages
2. Creá una nueva cartera desde el panel lateral
3. Agregá los activos de esa cartera
4. Abrí la app cada día para registrar el snapshot diario

## Activos soportados
- **Crypto**: BTC, ETH, SOL, ADA, BNB, XRP, etc. (vía CoinGecko)
- **US Equity / CEDEARs**: AAPL, MSFT, GOOGL, etc. (vía Yahoo Finance)
- **Acciones AR**: GGAL, YPFD, etc. (vía Yahoo Finance .BA)
- **Bonos AR**: AL30, GD30, etc. (vía Yahoo Finance .BA)
- **FCI / USD**: tipo de cambio blue (vía Bluelytics)
